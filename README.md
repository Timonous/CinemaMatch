# 🎬 CineMatch — Система персональных рекомендаций фильмов

Фронтенд-прототип для системы персональных и групповых рекомендаций фильмов.

## Страницы приложения

- **Авторизация** — вход и регистрация (мок)
- **Главная** — карточки фильмов с возможностью добавить в избранное или пропустить
- **Избранное** — список сохранённых фильмов с удалением
- **Профиль** — персональные данные, жанровые предпочтения

## Запуск проекта

### Шаг 1 — Установите Node.js
Скачайте и установите Node.js (v18+) с https://nodejs.org

### Шаг 2 — Установите зависимости
```bash
cd cinematch
npm install
```

### Шаг 3 — Запустите проект
```bash
npm start
```

Приложение откроется на http://localhost:3000

## Стек

- React 18
- Context API (глобальное состояние)
- Inline styles (без внешних CSS-фреймворков)
- Google Fonts (Playfair Display + DM Sans)
- Постеры фильмов: TMDB API (публичные изображения)
